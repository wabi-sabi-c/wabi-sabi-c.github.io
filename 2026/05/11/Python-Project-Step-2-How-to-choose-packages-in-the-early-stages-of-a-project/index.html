名称特点
TesseractGoogle 维护，老牌，但对中文和表格的识别效果一般

📋 其他包是怎么确定的？

需求我怎么搜得到什么结论
写Excel“Python 操作 Excel 写入 模板”主流就两个：openpyxl（操作.xlsx）和 xlsxwriter（只能写不能读）。我需要打开已有模板，所以选 openpyxl
接收图片上传去看 FastAPI 官方文档里的”File Upload”一章文档里写着安装 python-multipart
数据库操作复用之前 AI ChatFlow 项目经验sqlmodel + psycopg2-binary
配置管理同上，复用经验python-dotenv + pydantic-settings
前端网页极简 H5 页面可以直接用 FastAPI 返回 HTMLFastAPI 官方推荐 jinja2 渲染模板

✅ 所以以后可以怎么做？

在这个人工智能的时代，完全可以直接问任何AI，而是效率最高。专业的开发者也是这么干的：Google → 搜关键词 → 看文档 → 确定技术选型。
比如：

“AI，我要做一个手机拍照识别进货单、自动写入Excel台账的项目，帮我列一下需要哪些Python包，简单解释一下各自的用途。”

本质：“遇到问题 → 翻译成关键词 → 搜索 → 筛选 → 验证” 或者 直接问AI
