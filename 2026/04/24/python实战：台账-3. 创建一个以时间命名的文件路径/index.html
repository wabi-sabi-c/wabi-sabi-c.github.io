python实战：台账--创建一个以时间命名的文件路径

发表于 2026-04-24 分类于 python实战--台账 本文字数： 504 阅读时长 ≈ 2 分钟
  • 今天学习定义一个函数：创建一个以时间命名的文件路径
  1. 这个函数的功能是：自动生成一个以今天日期命名的文件路径。
  2. 它不会覆盖旧的文件、不会重复。
  3. 使用os.path.join(path, *paths)智能地合并一个或多个路径部分。 返回值将是 path 和所有 *paths 成员的拼接，其中每个非空部分后面都紧跟一个目录分隔符，最后一个除外。
  4. 因为没有创建xlsx文件只是生成文件名、文件路径，所有并不需要引入openpyxl 模块。
  5. 一定要return，返回值要不然这个函数没有任何东西可以调用
